## 1. Поточна постановка: CO2 ~ time (лінійний тренд)

### Дані
- n = 468, місячні спостереження 1959–1997
- y = концентрація CO2 (ppm), x = десятковий рік
- OLS: β₁ ≈ 1.308 ppm/рік, R² = 0.970

### Моменти залишків OLS
| Показник | Значення | Інтерпретація |
|----------|----------|---------------|
| gamma3 | +0.049 | симетричні ✓ |
| gamma4 | −0.625 | платикуртичні (помірно) |
| gamma6 | +1.987 | |
| g3_теор | 0.835 | PMM3 мав би дати −16.5% дисп. |

### Результати порівняння

**Bootstrap (g3_empirical = var(β̂₁) / var_OLS):**
| Метод | g3_emp | ∆ vs OLS |
|-------|--------|----------|
| OLS | 1.000 | — |
| PMM2 | 1.038 | −3.8% (гірше) |
| PMM3 | 1.039 | −3.9% (гірше) |
| **MLE-TN** | **0.958** | **+4.2% (краще)** |

**LOO-MSE (відносно OLS = 1.000):**
| Метод | rel_ols |
|-------|---------|
| OLS | 1.00000 |
| PMM2 | 1.00008 |
| PMM3 | 1.00000 |
| **MLE-TN** | **0.99989** |

### Висновок по поточній постановці
PMM3 **не виграє** попри gamma4 = −0.625 < 0. MLE-TN перемагає.

**Причина:** залишки від CO2 ~ time — це сезонна синусоїдальна компонента (~±3 ppm).
Розподіл синусоїди A·sin(2πt) за повний період — це **арксинусний розподіл**:

$$f(x) = \frac{1}{\pi\sqrt{A^2 - x^2}}, \quad x \in (-A, A)$$

Він **двомодовий** (маси на краях), симетричний, з gamma4 ≈ −1.5 (теоретично для
чистого синусоїда; тут −0.625 через шум вимірювань та нелінійність тренду).

MLE-TN (суміш двох усічених нормальних) природно моделює цей двомодовий розподіл.
PMM3 оцінює kappa з моментів залишків OLS — при gamma4 = −0.625 (вище емпіричного
порогу ≈ −0.70) шум оцінювання kappa нівелює теоретичну перевагу.

## 2. Гіпотеза: Сезонна модель + PMM3

### Мотивація
Якщо **включити сезонну компоненту** в регресійну модель, залишки більше не
містять синусоїди — вони стають "чистим" шумом вимірювань. Питання: яка природа
цього шуму? Чи є ще простір для PMM3?

### Варіанти розширеної моделі

**Варіант A — гармонічна регресія (пар Фур'є):**
```r
dat$sin1 <- sin(2 * pi * 12 * dat$x)
dat$cos1 <- cos(2 * pi * 12 * dat$x)
formula_s1 <- y ~ x + sin1 + cos1
```

**Варіант B — дві гармоніки (основна + обертон):**
```r
dat$sin2 <- sin(4 * pi * 12 * dat$x)
dat$cos2 <- cos(4 * pi * 12 * dat$x)
formula_s2 <- y ~ x + sin1 + cos1 + sin2 + cos2
```

**Варіант C — 11 місячних індикаторів:**
```r
dat$month <- as.factor(round((dat$x %% 1) * 12) %% 12)
formula_m <- y ~ x + month
```

### Очікувані сценарії для залишків після сезонної корекції

| Сценарій | gamma4 | Наслідок |
|----------|--------|----------|
| Чистий вимірювальний шум (Гаусів) | ≈ 0 | OLS оптимальний |
| Залишки мають важкі хвости (ENSO, вулкани) | > 0 | MLE-TN або PMM2 |
| Залишки платикуртичні (обмежені вимірювання) | < −0.7 | PMM3 виграє |
| Залишки асиметричні | |gamma3| ≥ 0.3 | PMM2 |

**Найбільш вірогідний результат:** після видалення сезонної компоненти залишки
будуть близькими до Гаусових (gamma4 ≈ 0), і OLS переможе всіх. Проте варто
перевірити — залишки можуть мати лептокуртичність через короткострокові аномалії
(вулканічна активність, Ель-Ніньо), що дало б перевагу MLE-TN з важкими хвостами.

### Ціннсть цього експерименту для статті

1. **Специфікація моделі визначає розподіл залишків** — той самий датасет CO2
   дає різні залишки залежно від включених регресорів.
2. **Проста лінійна модель** → бімодальні залишки (арксинус) → MLE-TN виграє.
3. **Правильно специфікована сезонна модель** → майже гаусові залишки → OLS виграє.
4. Це підтверджує, що PMM3 не є "срібною кулею" — він корисний тільки коли
   залишки дійсно мають **структурну** платикуртичність (gamma4 ≲ −0.7),
   як у iris-versicolor (обмежені вимірювання в межах виду).

### Зв'язок з репозиторієм PMM2-SARIMA (github.com/SZabolotnii/PMM2-SARIMA)

Репозиторій реалізує `sarima_pmm2()` — метод **PMM2** (2-й поліном, γ₃-based) для
SARIMA(p,d,q)(P,D,Q)s. **Це не PMM3** (4th cumulant). Архітектура:

- CSS fit → фіксовані залишки → дизайн-матриця X з AR/MA/SAR/SMA лагами
- PMM2 градієнт: `Z1 = m3·S² + (m4-m2²-2m3·Y)·S + ...` (Newton-Raphson, 3-5 ітер.)
- Для сезонних компонент — нелінійний `.pmm2_algorithm_nonlinear()` (файл `05_sarima_nonlinear_pmm2.R`)
- Умова ефективності: **|γ₃| > 0.5** (асиметрія); для CO2 γ₃=0.049 → **PMM2 не підходить**

**Різниця PMM2 vs PMM3 для сезонних моделей:**

| | PMM2-SARIMA (існує) | PMM3-SARIMA (гіпотетично) |
|--|---------------------|--------------------------|
| Цільовий кумулянт | γ₃ (асиметрія) | γ₄ (ексцес куртозис) |
| Умова ефективності | \|γ₃\| > 0.5 | γ₄ < −0.7 |
| CO2 linear trend | НЕ підходить (γ₃≈0) | Може (γ₄=−0.625, близько) |
| CO2 + seasonal | НЕ підходить | Залежить від нових γ₄ |

**Формула ефективності для SARIMA (з репозиторію):**
```
g_SARIMA = 1 - (γ₃² + λ_seasonal) / (2 + γ₄ + κ_seasonal)
```
де `λ_seasonal`, `κ_seasonal` — поправки на сезонну залежність.
Для PMM3-SARIMA (γ₃=0): `g = 1 - γ₄² / (6 + 9γ₄ + γ₆)` (та сама формула що в PMM3 регресії).

### PMM3-SARIMA: результати (2026-03-13)

**Реалізація:** `results/co2/R/pmm3_sarima.R`
**Алгоритм:** CSS-ML ініціалізація → фіксовані залишки → Newton-Raphson PMM3
**Скрипт аналізу:** `results/co2/scripts/09_co2_sarima_analysis.R`
**Результати:** `results/co2/output/sarima_comparison.csv`

#### Сезонна регресія y ~ x + sin(2πt) + cos(2πt)

| Метод | γ₃ | γ₄ | g3_теор | Примітка |
|-------|----|----|---------|----------|
| OLS | +0.408 | −0.580 | 0.899 | 10.1% потенціал |
| PMM3 | NaN | NaN | NA | Провал: γ₃=0.408 > 0.3 → PMM2 рекомендований |

**Увага:** формула `sin(2π*12*t)` де t = десятковий рік дає **нулі** для всіх місячних спостережень (cos → константа, колінеарна з перетином). Правильна формула: `sin(2π*(t%%1))`.

#### Ключовий висновок: чому PMM3-SARIMA не допомагає для CO2

Після подвійного диференціювання (d=1, D=1, s=12) та MA-фільтрації інновації **авіаційної моделі** стають **майже гаусовими**:
- γ₄ = +0.067 (≈0) → g3_теор = 0.9994 → PMM3 дає < 0.1% виграшу
- CSS-ML ≈ MLE ≈ PMM3 у параметрах (SARIMA добре специфікована)
- PMM3 RMSE дещо гірше: κ обчислюється з "перехідних" залишків першого сезону (γ₄≈−0.3), що вводить зміщення

**Загальний висновок для статті:**

| Модель | Залишки | γ₄ | Переможець |
|--------|---------|-----|------------|
| CO2 ~ time (лінійний тренд) | Синусоїда сезону | −0.625 | MLE-TN (арксинус) |
| ARIMA(0,1,1)(0,1,1)₁₂ | Інновації | +0.067 | CSS/MLE (гаусові) |
| y ~ x + sin + cos | Залишки після 1 гармоніки | −0.580, γ₃=+0.408 | PMM2 (асиметричні) |
| y ~ x + season (ідеально) | Чистий шум | ≈ 0 | OLS |

**Правильна специфікація моделі прибирає саме те, що давало б PMM3 перевагу.**

## 3. Зв'язок з іншими реальними даними проєкту

| Датасет | gamma4 | Переможець | Причина |
|---------|--------|------------|---------|
| iris-versicolor (SL~SW) | −0.966 | PMM3 | Обмежені вимірювання в межах виду |

| CO2 ~ time | −0.625 | MLE-TN | Бімодальна структура (арксинус) |
| CO2 ~ time+season (гіп.) | ≈ 0 ? | OLS? | Шум вимірювань після корекції |

**Узагальнення:** gamma4 < 0 — необхідна, але **не достатня** умова для PMM3.
Потрібна також **відсутність модальної структури** залишків (бімодальність → MLE-TN)
та **симетрія** (асиметрія → PMM2).
