## 2. Постановка A: Проста лінійна регресія `CO₂ ~ time`

### 2.1 Специфікація

```
y_t = β₀ + β₁ · t + ε_t
```

OLS-оцінки: β₀ = −2249.77, β₁ = 1.3075 ppm/рік, R² = 0.9695

### 2.2 Моменти залишків OLS

| Показник | Значення | Інтерпретація |
|----------|----------|---------------|
| γ₃ | +0.049 | Симетричні ✓ |
| γ₄ | **−0.625** | Платикуртичні |
| γ₆ | +1.987 | |
| g3_теор | **0.835** | PMM3 теоретично зменшує дисперсію на 16.5% |

**Діагноз:** `PMM3 (symmetric non-Gaussian errors)` — але γ₄ = −0.625 вище емпіричного порогу −0.70.

**Причина нестандартного розподілу:** залишки = сезонна синусоїда (~A·sin(2πt/12)). Розподіл синусоїди — **арксинусний**, симетричний і U-подібний:

$$f(x) = \frac{1}{\pi\sqrt{A^2 - x^2}}, \quad x \in (-A, A)$$

Теоретично: γ₄ = −1.5 для чистого синусоїда; спостережуване −0.625 = ефект вимірювального шуму + нелінійності тренду.

### 2.3 Порівняння методів (Bootstrap B=2000, seed=20260312)

| Метод | g3_emp = Var/Var_OLS | ∆ vs OLS |
|-------|---------------------|----------|
| OLS | 1.000 | — |
| PMM2 | 1.038 | −3.8% (гірше) |
| **PMM3** | **1.039** | **−3.9% (гірше!)** |
| **MLE-TN** | **0.958** | **+4.2% (краще)** |

### 2.4 LOO-MSE (n=468)

| Метод | LOO-MSE | rel_OLS |
|-------|---------|---------|
| OLS | 6.889 | 1.00000 |
| PMM2 | 6.889 | 1.00008 |
| PMM3 | 6.889 | 1.00000 |
| **MLE-TN** | **6.888** | **0.99989** |

### 2.5 Інтерпретація

**PMM3 програє попри γ₄ < 0.** Причина: арксинусний розподіл залишків є **структурно бімодальним** (маси на краях). MLE-TN (суміш двох усічених нормальних) природно підходить для цієї структури. PMM3 оцінює κ з моментів OLS-залишків — при γ₄ = −0.625 (вище порогу ≈ −0.70) шум оцінювання κ переважає теоретичний виграш.

## 3. Постановка B: ARIMA(0,1,1)(0,1,1)₁₂ — авіаційна модель

### 3.1 Специфікація

Стандартна модель для даних з лінійним трендом і сезонністю:

```
(1−B)(1−B¹²) y_t = (1 + θB)(1 + ΘB¹²) ε_t
```

Параметри: θ (MA1), Θ (SMA1). Подвійне диференціювання: d=1, D=1, s=12.

### 3.2 Оцінки параметрів

| Метод | θ (MA1) | Θ (SMA1) | σ² | AIC |
|-------|---------|----------|-----|-----|
| CSS-ML | −0.3501 | −0.8506 | 0.0826 | 178.16 |
| MLE | −0.3501 | −0.8507 | 0.0826 | 178.16 |
| **PMM3-SARIMA** | **−0.3462** | **−0.8011** | — | — |
| PMM3 зсув | +0.0038 | **+0.0495** | | |

PMM3 збігається за 3 ітерації Newton-Raphson. κ = −0.8516.

### 3.3 Моменти інновацій CSS-ML

| Показник | Значення | Інтерпретація |
|----------|----------|---------------|
| γ₃ | −0.017 | Симетричні ✓ |
| **γ₄** | **+0.067** | **≈ Гаусові** |
| g3_теор | 0.9994 | PMM3 < 0.1% виграш |

**Після подвійного диференціювання + MA-фільтрації інновації стають майже гаусовими.**

### 3.4 Hold-out RMSE (останні 24 місяці)

| Метод | RMSE |
|-------|------|
| CSS-ML | **0.34372** |
| MLE | **0.34372** |
| PMM3-SARIMA | 0.34675 (+0.9%, гірше) |

**PMM3-SARIMA програє**, оскільки κ обчислюється з перехідних залишків першого сезону (γ₄ ≈ −0.29), які ще зберігають сезонну структуру і є більш платикуртичними, ніж справжні інновації. Це вносить зміщення в оцінки θ та Θ.

## 5. Постановка D: Сезонна регресія `y ~ time + sin(2πt) + cos(2πt)`

### 5.1 Специфікація

```
y_t = β₀ + β₁ · t + β₂ · sin(2π · frac(t)) + β₃ · cos(2π · frac(t)) + ε_t
```

де `frac(t) = t %% 1` — дробова частина десяткового року.

**Технічна примітка:** формула `sin(2π·12·t)` = `sin(2π·integer)` ≡ 0 для всіх місячних спостережень → cos-регресор стає константою (колінеарна з перетином). Правильна формула: `sin(2π·(t%%1))`.

### 5.2 OLS-оцінки та діагностика

β₀ = −2256.25, β₁ = 1.311, β₂ = 2.772, β₃ = −0.389, R² = 0.987

| Показник | Значення | Інтерпретація |
|----------|----------|---------------|
| γ₃ | **+0.408** | **АСИМЕТРИЧНІ** (> 0.3!) |
| γ₄ | −0.580 | Платикуртичні |
| g3_теор | 0.899 | 10.1% потенційний виграш PMM3 |

**PMM3 провалюється** — попереджання `γ₃ = 0.408 > 0.3`, зсув = 0. PMM3 повертає OLS-оцінки.

### 5.3 Інтерпретація

Одна гармоніка захоплює лише основну синусоїдальну складову сезону. Залишки містять:
- Асиметричні аномалії (вулканічна активність: Agung 1963, El Chichón 1982, Pinatubo 1991)
- Нелінійності сезонного циклу (асиметрія зростання/спадання)

Результат: γ₃ = +0.408 → PMM2 більш доречний. Потрібні дві або більше гармоніки для симетричних залишків.

## 7. Ключові висновки для статті

### 7.1 Специфікація моделі визначає розподіл залишків

Один і той самий датасет CO₂ дає принципово різні розподіли залишків залежно від того, як специфікована модель:

```
Неповна специфікація (CO₂ ~ time):
  залишки = сезонний синусоїд → арксинус → MLE-TN виграє

Правильна специфікація (ARIMA airline):
  залишки = інновації → ≈ Гаусові → OLS/CSS/MLE оптимальні

Часткова специфікація (y ~ time + 1 гармоніка):
  залишки = асиметричний шум → PMM2 потрібен
```

### 7.2 Умови ефективності PMM3

PMM3 дає реальний виграш лише при одночасному виконанні ТРЬОХ умов:
1. **γ₃ ≈ 0** — симетричні залишки (|γ₃| < 0.3)
2. **γ₄ ≲ −0.70** — достатньо платикуртичні (вище порогу − PMM3 програє)
3. **Відсутність модальної структури** — бімодальні залишки → MLE-TN

Для CO₂:
- Постановка A: γ₄ = −0.625 (вище порогу −0.70) + бімодальна структура → обидві умови 2 і 3 порушені
- Постановка B/C: γ₄ ≈ 0 → умова 2 порушена

### 7.3 PMM3-SARIMA: реалізація та обмеження

PMM3-SARIMA (реалізований у `results/co2/R/pmm3_sarima.R`):
- **Алгоритм:** CSS-ML ініціалізація → фіксовані залишки (EstemPMM-style) → Newton-Raphson
- **Збіжність:** 3 ітерації за 1e-10
- **Обмеження:** κ обчислюється з перехідних залишків першого сезону, які більш платикуртичні за справжні інновації → зміщення оцінок при майже-гаусових даних

**Висновок:** PMM3-SARIMA потрібен лише для часових рядів, де **після диференціювання** та **після всіх MA-фільтрів** інновації залишаються платикуртичними (γ₄ ≲ −0.7). Для CO₂ такої ситуації не виникає.

### 7.4 Контраст з PMM2-SARIMA (github.com/SZabolotnii/PMM2-SARIMA)

| Властивість | PMM2-SARIMA | PMM3-SARIMA |
|-------------|-------------|-------------|
| Цільовий кумулянт | γ₃ (3-й порядок) | γ₄ (4-й порядок) |
| Умова ефективності | \|γ₃\| > 0.5 | γ₄ < −0.7 |
| Підходить для CO₂ | Ні (γ₃ ≈ 0.05) | Ні (γ₄ ≈ 0 після ARIMA) |
| Практичний приклад | Фінансові ряди (скошені) | Виміри з обмеженим діапазоном |
