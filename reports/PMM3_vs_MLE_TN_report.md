## 1. Вступ та постановка задачі

Розглядається модель лінійної регресії:

$$y = X\beta + \varepsilon, \quad \varepsilon \sim \text{симетричний розподіл з } E(\varepsilon)=0$$

Порівнюються три методи оцінювання вектора параметрів β:

| Метод | Скорочення | Тип | Оцінює |
|---|---|---|---|
| Метод найменших квадратів | **OLS** | Параметричний (Гаусс) | β |
| Метод поліноміальної максимізації, S=3 | **PMM3** | Напівпараметричний | β |
| Метод максимальної правдоподібності для TN | **MLE-TN** | Параметричний (TN) | β, η, λ |

**Мета дослідження:** визначити умови, за яких кожен метод є найефективнішим, та з'ясувати залежність ефективності від:
- параметра форми λ TN-розподілу (і, відповідно, коефіцієнта ексцесу γ₄);
- обсягу вибірки n;
- відстані між модами, нормованої на дисперсію (для бімодальних розподілів);
- правильності специфікації моделі розподілу.

## 2. Математичні основи методів

### 2.1 OLS — Метод найменших квадратів

Мінімізує суму квадратів залишків:

$$\hat{\beta}_{OLS} = (X^T X)^{-1} X^T y$$

Оптимальний при гауссових похибках. Для негауссових симетричних розподілів — консистентний, але неефективний.

### 2.2 PMM3 — Метод поліноміальної максимізації (S=3)

**Оцінює:** тільки β (p параметрів).
**Не оцінює:** параметри розподілу похибок (η, λ, σ тощо).

**Алгоритм:**

1. OLS-старт: β̂₀ = β̂_OLS, обчислення моментів залишків m₂, m₄, m₆
2. Фіксація момент-залежної константи:
   $$\kappa = \frac{m_6 - 3 m_4 m_2}{m_4 - 3 m_2^2}$$
3. Newton-Raphson ітерації:
   - Score: $Z_{1v} = \varepsilon_v(\kappa - \varepsilon_v^2)$, де $\varepsilon_v = y_v - X_v^T \beta$
   - Score-вектор: $Z_p = \sum_v Z_{1v} \cdot x_{vp}$
   - Якобіан: $J_{pq} = \sum_v (3\varepsilon_v^2 - \kappa) \cdot x_{vp} x_{vq}$, тобто $J = X^T \text{diag}(3\varepsilon^2 - \kappa) X$
   - Оновлення: $\beta \leftarrow \beta - J^{-1} Z$
4. Збіжність: $\|Q\|_2 < 10^{-6}$ (зазвичай 3–5 ітерацій)
5. Захист від розбіжності: якщо $\|\beta - \beta_{OLS}\| > 10$, повертається OLS

**Теоретичний коефіцієнт ефективності:**

$$g_3 = \frac{\text{Var}(\hat{\beta}_{PMM3})}{\text{Var}(\hat{\beta}_{OLS})} = 1 - \frac{\gamma_4^2}{6 + 9\gamma_4 + \gamma_6}$$

де γ₄ = μ₄/μ₂² − 3 (ексцес), γ₆ = μ₆/μ₂³ − 15(μ₄/μ₂²) + 30.

**PMM3 не потребує знання функціональної форми розподілу** — використовує лише моменти 2-го, 4-го та 6-го порядку.

### 2.3 MLE-TN — Максимальна правдоподібність для TN

**Оцінює:** β, η (масштаб), λ (форма) — разом p+2 параметри.

Щільність TN (формула 9, Salinas et al. 2023):

$$f(z|\lambda) = \frac{1}{\sqrt{2\pi}} \exp\left(-\frac{z^2}{2} - \frac{\lambda^2}{2}\right) \cdot \cosh(\lambda z)$$

Логарифмічна правдоподібність максимізується L-BFGS-B з OLS-стартом:
- β-старт: OLS-оцінки
- η-старт: log(sd(ê_OLS))
- λ-старт: 1.0 (фіксоване значення за замовчуванням)

**MLE-TN припускає, що розподіл похибок є TN** — при правильній специфікації досягає межі Крамера-Рао, при misspecification може давати некоректні оцінки.

### 2.4 Ключова відмінність: що оцінюється

| Метод | Параметри | Кількість | Потрібна форма розподілу? |
|---|---|---|---|
| **OLS** | β | p | Ні (оптимальний лише при Гаусс) |
| **PMM3** | β | p | **Ні** (будь-який симетричний) |
| **MLE-TN** | β, η, λ | p + 2 | **Так** (тільки TN) |

### 3.2 Бімодальність та відстань між модами

Для TN(λ) моди визначаються рівнянням $\tanh(\lambda x^*) = x^*/\lambda$. Бімодальність виникає при λ > 1.

| λ | Бімодальний? | 2x*/σ (нормована відстань) |
|---|---|---|
| 0.0–1.0 | Ні | 0 (одна мода) |
| 1.2 | Так | 1.281 |
| 1.5 | Так | 1.623 |
| 2.0 | Так | 1.788 |
| 3.0 | Так | 1.897 |
| 5.0 | Так | 1.961 |

Нормована відстань між модами асимптотично наближається до 2σ при λ → ∞. Зростання відстані між модами корелює зі збільшенням |γ₄| та підвищенням ефективності PMM3.

## 6. Експеримент 3: Залежність ефективності від коефіцієнта ексцесу γ₄

**Умови:** M = 600 реплікацій, n = 200. Досліджено 13 розподілів з різними γ₄.

**Гіпотеза:** ефективність PMM3 визначається переважно коефіцієнтом ексцесу γ₄, незалежно від конкретної форми розподілу. Коефіцієнт ексцесу, в свою чергу, визначається відстанню між модами, нормованою на дисперсію.

де BimixN(d) — суміш 0.5·N(−d, 1) + 0.5·N(+d, 1), нормована до Var = 1.

### Перевірка інваріантності ефективності відносно γ₄

Розподіли з **однаковим γ₄** дають **практично однаковий ARE_PMM3**:

| Пара розподілів | γ₄ | ARE_PMM3 (перший) | ARE_PMM3 (другий) |
|---|---|---|---|
| TN(λ=1.0) vs BimixN(d=1) | −0.50 | 1.08 | 1.07 |
| TN(λ=2.0) vs BimixN(d=2) | −1.28 | 2.55 | 3.01 |
| TN(λ=3.0) vs BimixN(d=3) | −1.62 | 7.12 | 6.89 |

Невеликі розбіжності зумовлені статистичним шумом (M = 600) і незначними відмінностями в γ₆.

### Зв'язок γ₄ з відстанню між модами

Для бімодальної суміші 0.5·N(−d, σ) + 0.5·N(+d, σ):
- Нормована відстань між модами: $\delta = 2d / \sqrt{d^2 + \sigma^2}$
- γ₄ = (3σ⁴ + 6σ²d² + d⁴)/(d² + σ²)² − 3

Зростання δ (ширше розведення мод) → більш від'ємний γ₄ → менший g₃ → вищий ARE_PMM3.

Для TN(λ): при λ > 1 також спостерігається монотонна залежність: λ↑ → |γ₄|↑ → 2x*/σ↑ → ARE↑.

### Висновки з Експерименту 3

1. **Підтверджено:** ефективність PMM3 визначається переважно коефіцієнтом ексцесу γ₄. Розподіли з однаковим γ₄ (TN та BimixN) дають практично однаковий ARE_PMM3.

2. **γ₆ — поправковий член.** Він входить у формулу g₃, але для платикуртичних розподілів γ₆ і γ₄ зростають разом, тому γ₄ є домінантним фактором.

3. **Лептокуртичні розподіли (γ₄ > 0):** PMM3 дає скромний виграш (ARE ≈ 1.05–1.08 для Logistic і t(10)). MLE-TN ≈ OLS — модель TN не може описати важкі хвости.

4. **PMM3 нейтральний до форми розподілу** — це напівпараметричний метод, що працює однаково добре для TN, рівномірного, бімодальної суміші — для будь-якого симетричного розподілу з заданим γ₄.

## 7. Експеримент 4: Адаптивний PMM3 (невідомі параметри розподілу)

### Висновки з Експерименту 4

1. **PMM3-fixed ≈ PMM3-adaptive при TN.** Різниця в межах статистичного шуму. Теоретично обидва асимптотично еквівалентні, оскільки OLS-моменти є консистентними оцінками.

2. **Для рівномірного розподілу: адаптивний PMM3 дещо кращий** (на 0.03–0.15 за ARE). Основна перевага — стійкість: адаптивний режим не «засмічується» невірними OLS-моментами при сильно ненормальних даних.

3. **Практична рекомендація:** адаптивний режим рекомендовано тільки при підозрі на суттєве відхилення від передбачуваної форми розподілу. Для більшості задач fixed-режим достатній.

## 8. Експеримент 5: Робастність при misspecification

**Питання:** що відбувається, коли реальний розподіл похибок не є TN?

### Результати (M = 500, n = 200)

| DGP (реальний розподіл) | γ₄ | ARE PMM3 | ARE MLE-TN | Хто кращий? |
|---|---|---|---|---|
| **Uniform** (−√3, √3) | −1.20 | **2.88** | 1.64 | **PMM3 >> MLE-TN** |
| **Logistic** (0, √3/π) | +1.20 | **1.12** | 1.00 | PMM3 > MLE-TN |
| **t(10)** | +1.00 | 1.03 | 1.00 | PMM3 ≈ MLE-TN |

### Інтерпретація

1. **Uniform (платикуртичний, γ₄ = −1.2):** PMM3 дає ARE ≈ 2.9 — відчуває від'ємний ексцес і адаптує ваги через κ. MLE-TN підбирає параметри TN під рівномірний розподіл — неправильна сімʼя, тому виграш скромний (1.6).

2. **Logistic (лептокуртичний, γ₄ = +1.2):** PMM3 дає скромний виграш ≈ 1.1. MLE-TN ≈ OLS — модель TN не може описати важкі хвости логістичного розподілу.

3. **t(10) (слабо лептокуртичний, γ₄ = +1.0):** обидва методи ≈ OLS. Розподіл t(10) дуже близький до гауссового.

4. **Висновок:** PMM3 робастніший за MLE-TN при misspecification. Коли розподіл невідомий — PMM3 є безпечнішим вибором.

### 9.6 Новий ключовий висновок: емпіричний поріг PMM3

Зведені дані виявляють чіткий **розрив між теоретичним та емпіричним порогами ефективності PMM3**:

| Діапазон γ₄ | Теорет. покращення | Емпіричний boot_g₃ | Висновок |
|---|---|---|---|
| γ₄ < −0.7 | > 30% | **< 0.75** (PMM3 виграє) | PMM3 рекомендовано |
| −0.7 ≤ γ₄ < −0.2 | 8–30% | ≈ 1.0 або > 1.0 | PMM3 ≈ OLS або гірше |
| γ₄ ≥ −0.2 або γ₃ ≠ 0 | < 8% | > 1.0 | OLS або PMM2 |

**Чому емпіричний поріг γ₄ ≲ −0.7 вищий за теоретичний γ₄ < 0?**

Теоретична формула g₃ = 1 − γ₄²/(6 + 9γ₄ + γ₆) припускає **відоме** κ. На практиці κ оцінюється з n OLS-залишків з похибкою Var(κ̂) ∝ 1/n. При малому |γ₄| корисний сигнал (величина перерозподілу ваг) менший за шум оцінки κ. Тільки при γ₄ ≲ −0.7 сигнал достатньо великий, щоб подолати цей шум при типових n = 30–100.

**Практичне правило:** застосовуйте PMM3, якщо `resid_diagnostics()` показує γ₄ < −0.7 і |γ₃| < 0.3. При γ₄ ∈ (−0.7, 0) — PMM3 ≈ OLS на фінітних вибірках.

### 9.7 Висновки по реальних даних

1. **Єдиний датасет, де PMM3 чітко перемагає — iris-versicolor** (γ₄ = −0.97, bounded/uniform залишки, Span/SD = 3.58 ≈ 3.46). Boot g₃_emp = 0.570 узгоджується з теорією (g₃_theор = 0.483).

2. **Емпіричний поріг γ₄ ≲ −0.7**: нижче якого PMM3 стабільно ефективний, вище — переваги не спостерігається на реальних n=30–300.

3. **MLE-TN ненадійне при n < 50**: 6–18% NA у bootstrap на datасетах із n=31–50 через нестабільність L-BFGS-B оптимізатора. LOO-MSE більш надійно відображає реальну точність.

4. **Системна діагностика spрацьовує**: для biaxial fatigue (γ₃ = −0.47 → PMM2), для iris-versicolor (γ₄ = −0.97 → PMM3) — обидва рекомендовані методи відповідають дійсності.

5. **LOO-MSE відрізняється < 2%** на всіх датасетах крім iris-versicolor — у завданнях передбачення при n = 30–300 жоден метод не дає значущого виграшу над OLS, якщо γ₄ > −0.7.

## 10. Загальні висновки

### 10.1 Коли використовувати який метод

| Сценарій | Рекомендований метод | Обґрунтування |
|---|---|---|
| Розподіл відомий = TN, n ≥ 50 | **MLE-TN** | Максимальна ефективність (межа Крамера-Рао) |
| Розподіл відомий = TN, n < 30 | **PMM3 або MLE-TN** | При λ ≤ 1.5 обидва порівнянні; при λ > 2 MLE-TN кращий |
| Розподіл невідомий, симетричний | **PMM3** | Не потребує специфікації, робастний |
| Розподіл невідомий, можливо асиметричний | **PMM2** | PMM3 оптимізовано для симетричних розподілів |
| Розподіл гауссовий або близький | **OLS** | Найпростіший, жодних додаткових параметрів |
| γ₄ ≈ 0 і/або γ₃ ≠ 0 (реальні дані) | **OLS або PMM2** | PMM3/MLE-TN не дадуть виграшу; спочатку діагностика |

### 10.2 Фактори, що визначають ефективність PMM3

1. **Коефіцієнт ексцесу γ₄** — домінантний фактор. Чим більш від'ємний γ₄ (сильніший платикуртоз), тим вища ефективність PMM3 (нижче g₃).

2. **Відстань між модами** — для бімодальних розподілів γ₄ і нормована відстань між модами 2x*/σ монотонно пов'язані. Ширше розведення мод → більш від'ємний γ₄ → вищий ARE.

3. **Обсяг вибірки n** — при малих n (< 30) PMM3 ≈ MLE-TN через «вартість» оцінки додаткових параметрів η, λ. При n > 50 MLE-TN виходить на асимптотичну перевагу (при правильній специфікації).

4. **Правильність специфікації** — MLE-TN ефективний тільки при правильно заданому розподілі. PMM3 працює для будь-якого симетричного розподілу, що робить його **безпечнішим вибором у реальних застосуваннях**.

### 10.3 Кількісні підсумки

- **PMM3 дає ~80–90% від виграшу MLE-TN при правильній TN-специфікації**, не знаючи форми розподілу.
- **При misspecification PMM3 значно перевершує MLE-TN** (ARE PMM3 ≈ 2.9 vs ARE MLE-TN ≈ 1.6 для рівномірного розподілу).
- **Теоретична формула g₃ = 1 − γ₄²/(6 + 9γ₄ + γ₆) підтверджена емпірично** для TN та бімодальних сумішей з однаковим γ₄.
- **100% збіжність** обох методів у всіх Monte Carlo експериментах (M до 1000, n від 20 до 1000).
- **На реальних даних (7 датасетів):** PMM3 перемагає лише при γ₄ ≲ −0.7. Iris-versicolor (γ₄=−0.97): g₃_emp=0.570 (43% виграш), PMM3 > MLE-TN. Всі інші датасети з γ₄ ∈ (−0.6, 0): PMM3 ≈ OLS або гірше — **емпіричний поріг γ₄ ≲ −0.7** визначено як практичний критерій застосування PMM3.

### 10.4 Відкриті питання для подальшого дослідження

1. **Множинна регресія** (p > 2): як масштабується ефективність з кількістю регресорів?
2. **Асиметричні розподіли:** повне порівняння PMM2 vs PMM3 vs MLE-TN при γ₃ ≠ 0.
3. **Гетероскедастичність:** поведінка PMM3 при порушенні припущення про однорідність дисперсій.
4. **Виправлення LOO dispatch-конфлікту** між PMM3fit і PMM2fit predict-методами.
