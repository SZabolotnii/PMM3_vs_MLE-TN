## Анотація

Розглядається задача оцінювання вектора параметрів лінійної регресії при симетричних негауссових похибках з двочастинним нормальним розподілом (Two-piece Normal, TN). Порівнюються три методи: метод найменших квадратів (МНК/OLS), метод максимального правдоподібності для TN-розподілу (MLE-TN) та метод поліноміальної максимізації третього ступеня (PMM3). MLE-TN є повністю параметричним і потребує правильної специфікації функціональної форми розподілу, тоді як PMM3 — напівпараметричний і ґрунтується лише на моментах 2-го, 4-го та 6-го порядків. Теоретично показано, що PMM3 досягає до 95.5% зниження дисперсії порівняно з MНК при λ = 5 (TN-параметр форми), а MLE-TN забезпечує до 28× перевагу при великих λ. Проведено Монте-Карло дослідження (M = 500, n ∈ {46, 100, 200, 500}), яке підтвердило теоретичні передбачення: емпіричний коефіцієнт g₃ збігається з теоретичним при n ≥ 100. Показано, що PMM3 є робастним при неправильній специфікації (ARE ≈ 1.0), тоді як MLE-TN може бути неефективним або нестійким. На реальних даних (ірис Фішера, n = 50, γ₄ = −0.966) PMM3 досягає 43% зниження дисперсії за бутстрепом та покращення LOO-CV на 2.3% порівняно з МНК. Визначено емпіричний поріг ефективності PMM3: γ₄ ≲ −0.7.

**Ключові слова:** метод поліноміальної максимізації, лінійна регресія, двочастинний нормальний розподіл, метод максимального правдоподібності, ефективність оцінювання, робастність, Монте-Карло

## 1. Вступ

### 1.1 Мотивація

Класичний метод найменших квадратів (МНК) є оптимальним лише при гауссовому розподілі похибок. На практиці в економетрії, інженерії та природничих науках похибки часто мають симетричні, але негауссові розподіли — зокрема платикуртичні (від'ємний ексцес, γ₄ < 0) або лептокуртичні (γ₄ > 0). Ігнорування цього факту призводить до неефективного оцінювання.

Одним із важливих класів симетричних негауссових розподілів є **двочастинний нормальний розподіл** (Two-piece Normal, TN), уведений Джоном (1982) та досліджений Саліносом та ін. (2023). TN охоплює широкий спектр форм: від майже гауссового (λ ≈ 0) до різко бімодального (λ ≫ 1.2), при цьому завжди залишаючись симетричним (якщо масштабовано відповідним чином).

Існують два принципово різні підходи до підвищення ефективності оцінювання при TN-похибках:

1. **Параметричний (MLE-TN)**: оцінює всі параметри (β, η, λ) спільно, використовуючи повну правдоподібність. Оптимальний при правильній специфікації, але чутливий до неправильної форми розподілу.
2. **Напівпараметричний (PMM3)**: використовує лише вибіркові моменти для корекції МНК-оцінки. Не потребує знання форми розподілу, але дає меншу теоретичну ефективність.

### 1.2 Огляд літератури

**Метод поліноміальної максимізації (МПП/PMM)** розроблений Кунченком і Легою (1992) для оцінювання параметрів стаціонарних випадкових послідовностей. Метод ґрунтується на максимізації поліноміальної функції від спостережень і дозволяє врахувати старші кумулянти розподілу.

Заболотній та ін. (2018) застосували PMM2 (S = 2) до лінійної регресії з асиметричними похибками і показали теоретичну перевагу: g₂ = 1 − c₃²/(2 + c₄), де c₃, c₄ — нормовані кумулянти. PMM3 (S = 3) є природним узагальненням для симетричних розподілів (c₃ = 0), де g₂ = 1 і вдосконалення можливе лише через кумулянти 4-го та 6-го порядків.

Для часових рядів PMM2 застосовано до авторегресійних (Zabotniy та ін., 2022) та ковзних (2023) моделей, а також до ARIMA (Zabolotnii, 2025).

**Двочастинний нормальний розподіл** у контексті лінійної регресії досліджено Саліносом та ін. (2023), які запропонували MLE-оцінювання та навели таблиці кумулянтів для різних λ.

### 1.3 Внесок роботи

Дана робота є першою, яка систематично порівнює PMM3 і MLE-TN для лінійної регресії з TN-похибками. Основні результати:

1. **Теоретичне порівняння**: аналітичні вирази для g₃(λ) та ARE(MLE-TN/OLS)(λ).
2. **Монте-Карло підтвердження**: емпіричні ARE, g₃ для λ ∈ {0.5, 1, 1.5, 2, 3, 5}, n ∈ {46, 100, 200, 500}.
3. **Аналіз робастності**: поведінка PMM3 та MLE-TN при неправильній специфікації.
4. **Емпіричний поріг**: γ₄ ≲ −0.7 як практичне правило для вибору PMM3 над МНК.
5. **Реальні дані**: підтвердження теорії на прикладі ірисів Фішера.

## 2. Методологія

### 2.1 Модель лінійної регресії

Розглядається лінійна регресійна модель:

$$y_v = \mathbf{x}_v^T \boldsymbol{\beta} + \varepsilon_v, \quad v = 1, \ldots, n$$

де y_v — відгук, **x**_v ∈ ℝᵖ — вектор регресорів, **β** ∈ ℝᵖ — вектор невідомих параметрів, ε_v — незалежні однаково розподілені похибки.

**Припущення (для PMM3):** ε_v ~ симетричний розподіл з E(ε) = 0, скінченними моментами до 6-го порядку.

**Припущення (для MLE-TN):** ε_v ~ TN(0, η, λ).

### 2.2 Двочастинний нормальний розподіл (TN)

TN-розподіл з нульовим центром, масштабом η > 0 та параметром форми λ > 0 має щільність (Salinas et al., 2023, формула 9):

$$f(z \mid \lambda) = \frac{1}{\sqrt{2\pi}} \exp\!\left(-\frac{z^2}{2} - \frac{\lambda^2}{2}\right) \cosh(\lambda z), \quad z \in \mathbb{R}$$

**Ключові властивості:**
- Симетричний відносно нуля при η = 1 (або нормованих даних)
- λ → 0: наближається до N(0,1)
- λ = 1/√2 ≈ 0.707: унімодальний-бімодальний поріг
- λ > 1/√2: бімодальний (два симетричних піки)
- Коефіцієнт ексцесу: γ₄ строго від'ємний для λ > 0 (платикуртичний)

**Централізовані моменти та кумулянти** обчислюються рекурентно:

$$\mu_{2k} = \int_{-\infty}^{\infty} z^{2k} f(z|\lambda)\, dz$$

Нормований ексцес:

$$\gamma_4 = \frac{\mu_4}{\mu_2^2} - 3 < 0 \quad \text{для всіх } \lambda > 0$$

Таблиця 1 показує значення γ₄ та теоретичного g₃ для репрезентативних λ.

### 2.3 MLE-TN — Метод максимального правдоподібності

MLE-TN оцінює вектор (**β**, η, λ) спільно, максимізуючи логарифм правдоподібності:

$$\ell(\boldsymbol{\beta}, \eta, \lambda \mid \mathbf{y}) = \sum_{v=1}^{n} \log f\!\left(\frac{y_v - \mathbf{x}_v^T\boldsymbol{\beta}}{\eta} \;\middle|\; \lambda\right) - n \log \eta$$

Оптимізація виконується методом L-BFGS-B зі стартовими точками:
- **β**₀ = **β**̂_OLS (МНК-оцінки)
- η₀ = exp(log SD(ê_OLS))
- λ₀ = 1.0

**Переваги:** при правильній специфікації досягає межі Крамера–Рао; ARE(MLE-TN/OLS) може бути >> 1.

**Недоліки:** p + 2 параметри; чутливість до неправильної специфікації; числова нестійкість при малих n і великих λ.

### 2.4 PMM3 — Метод поліноміальної максимізації (S = 3)

PMM3 оцінює лише **β** (p параметрів), використовуючи момент-коригований score:

**Алгоритм:**

1. **Старт:** **β**⁽⁰⁾ = **β**̂_OLS. Обчислити вибіркові центральні моменти залишків m₂, m₄, m₆.

2. **Фіксована константа:**
   $$\kappa = \frac{m_6 - 3\,m_4\,m_2}{m_4 - 3\,m_2^2}$$
   (Якщо |m₄ − 3m₂²| < 10⁻¹⁰, повернути **β**̂_OLS — Гауссовий запасний варіант.)

3. **Newton–Raphson ітерації:**
   - Score для спостереження v: $Z_{1v} = \varepsilon_v(\kappa - \varepsilon_v^2)$
   - Вектор score: $\mathbf{Z} = X^T \mathbf{Z}_1$ (де $(\mathbf{Z}_1)_v = Z_{1v}$)
   - Якобіан: $J = X^T \operatorname{diag}(3\varepsilon^2 - \kappa)\, X$
   - Оновлення: **β** ← **β** − J⁻¹ **Z**

4. **Критерій збіжності:** ‖**Z**‖₂ < 10⁻⁶ (зазвичай 3–5 ітерацій)

5. **Захист від розбіжності:** якщо ‖**β** − **β**̂_OLS‖ > 10, повернути **β**̂_OLS.

**Теоретичний коефіцієнт ефективності:**

$$g_3 = \frac{\operatorname{Var}(\hat{\boldsymbol{\beta}}_{PMM3})}{\operatorname{Var}(\hat{\boldsymbol{\beta}}_{OLS})} = 1 - \frac{\gamma_4^2}{6 + 9\gamma_4 + \gamma_6}$$

де γ₄ = μ₄/μ₂² − 3, γ₆ = μ₆/μ₂³ − 15(μ₄/μ₂²) + 30 (нормовані кумулянти 4-го та 6-го порядків).

**PMM3 не потребує знання функціональної форми розподілу** — коефіцієнт g₃ залежить лише від кумулянтів, які оцінюються безпосередньо з вибірки.

### 2.5 Асимптотична нормальність PMM3

При стандартних умовах регулярності (незалежні, однаково розподілені похибки з скінченними моментами до 6-го порядку) оцінка PMM3 є **консистентною та асимптотично нормальною**:

$$\sqrt{n}(\hat{\boldsymbol{\beta}}_{PMM3} - \boldsymbol{\beta}) \xrightarrow{d} \mathcal{N}\!\left(\mathbf{0},\; g_3 \cdot \sigma^2 (X^TX/n)^{-1}\right)$$

де g₃ < 1 при γ₄ < 0 — коефіцієнт ефективності PMM3.

Кова́ріаційна матриця PMM3 аналогічна МНК-матриці, але зменшена в (1/g₃) разів:

$$\operatorname{Var}(\hat{\boldsymbol{\beta}}_{PMM3}) = g_3 \cdot \operatorname{Var}(\hat{\boldsymbol{\beta}}_{OLS})$$

**Наслідки:**
1. При γ₄ = 0 (Гаусс) g₃ = 1, тобто PMM3 ≡ МНК асимптотично.
2. При γ₄ < 0 (платикуртичний) g₃ < 1 — PMM3 ефективніший за МНК.
3. Стандартні помилки PMM3 = √g₃ × SE(OLS) — можна обчислити бутстрепом.
4. Консистентність не залежить від форми розподілу — лише від нульового середнього.

*(Докладніший вивід: Kunchenko & Lega, 1992; Zabolotnii et al., 2018.)*

### 2.6 Порівняльна характеристика методів

| Метод | Параметри | Кількість | Потрібна форма? | Ефективність при TN |
|-------|-----------|-----------|-----------------|---------------------|
| OLS | **β** | p | Ні (оптимальний при Гаусс) | Базова (ARE = 1) |
| PMM3 | **β** | p | Ні (будь-який симетричний) | g₃ < 1 при γ₄ < 0 |
| MLE-TN | **β**, η, λ | p + 2 | Так (тільки TN) | Крамер-Рао при TN |

## 4. Монте-Карло дослідження

### 4.1 Дизайн

**Модель даних:**
$$y_v = 1 + \beta_1 x_v + \varepsilon_v, \quad \varepsilon_v \sim \text{TN}(0, 1, \lambda)$$

де x_v ~ N(0, 1), β₁ = 1.0 (оцінюваний параметр).

**Параметри симуляції:**
- M = 500 реплікацій
- n ∈ {46, 100, 200, 500}
- λ ∈ {0.5, 1.0, 1.5, 2.0, 3.0, 5.0}
- Генератор: seed = 20260312

**Метрики:**
- Зміщення: Bias = $\overline{\hat{\beta}_1} - 1$
- Дисперсія: Var = $\widehat{\operatorname{Var}}(\hat{\beta}_1)$
- MSE = Bias² + Var
- ARE = Var(OLS)/Var(метод)
- Емпіричний g₃ = Var(PMM3)/Var(OLS)

### 4.3 Інтерпретація

**Ключові спостереження:**

1. **PMM3 суттєво перевершує МНК** при λ ≥ 1.5: ARE від 1.5× до 23.8× — підтверджено теорію.
   При λ = 0.5 (γ₄ = −0.08, близько до Гаусс) PMM3 ≈ МНК (g₃_emp ≈ 1.00).

2. **MLE-TN ще ефективніший за PMM3** при λ ≥ 1.5, бо використовує повну правдоподібність.
   Але MLE-TN оцінює p+2 параметри, що при малих n призводить до надмірної параметризації.

3. **g₃_emp ≈ g₃_теор** з точністю ±10% при n ≥ 100 — гіпотеза H1 підтверджена.
   Найбільше відхилення при малих n (n = 46) і великих λ — нормально для скінченних вибірок.

4. **Збіжність PMM3 100%** для всіх 24 блоків (у т.ч. бімодальні λ = 3, 5) — алгоритм стійкий.

5. **При λ = 5, n = 100**: ARE(PMM3) = 23.8× vs ARE(MLE-TN) = 27.4× — PMM3 відстає від
   MLE-TN лише на 15%, але не потребує знання форми розподілу. При λ = 5 (бімодальний,
   γ₄ = −1.85) обидва методи на порядок ефективніші за МНК.

## 5. Реальні дані — Ірис Фішера (Iris versicolor)

### 5.1 Набір даних

Використовується підвибірка ірисів Фішера (Fisher, 1936): вид Iris versicolor, n = 50 спостережень. Модель:

$$\text{Sepal.Length}_v = \beta_0 + \beta_1 \cdot \text{Sepal.Width}_v + \varepsilon_v$$

**Діагностика залишків МНК** (гістограма: **Рис. 5**, Fig5_iris_residuals.png):
- n = 50
- γ̂₃ = 0.273 (незначна асиметрія)
- γ̂₄ = −0.966 (виражений від'ємний ексцес)
- Порівняння з діапазоном/SD: span/SD ≈ 3.58 ≈ 3.46 (Uniform!) — свідчить про обмежений діапазон вимірювань

**Теоретична ефективність:** g₃_теор = 0.517 (51.7% зниження дисперсії при TN з γ₄ = −0.966)

**Діагностичний висновок:** γ₄ = −0.966 < −0.7 → рекомендується PMM3.

| PMM3 | 0.797 | **0.2022** | **0.570** |
| MLE-TN | 0.753 | 0.2109 | 0.746 |

*g₃_emp = Var(метод)/Var(OLS) за бутстрепом з B = 2000 реплікацій.*

### 5.3 Інтерпретація

1. **PMM3 найкращий** за обома критеріями: LOO-MSE = 0.2022 < OLS = 0.2070 < MLE-TN = 0.2109.
2. **g₃_emp = 0.570**: 43% зниження дисперсії (порівняно з теоретичним 51.7% — розумна відповідність для n = 50).
3. **MLE-TN найгірший** попри наявність γ₄ < 0: MLE-TN оцінює додаткові параметри η, λ при лише n = 50, що призводить до надмірної параметризації.
4. **Практичний висновок:** при n = 50 і γ₄ ≈ −0.97, PMM3 є оптимальним вибором.

### 5.4 Негативні приклади (для повноти)

Для перевірки відтворюваності результатів розглянуто набір даних `whiteside` (пакет `MASS`):
газоспоживання будинку як функція температури до/після утеплення.

**whiteside: Gas ~ Temp + Insul (n = 56, пакет MASS):**
- γ̂₃ = −0.171 (незначна асиметрія)
- γ̂₄ = −0.431 (помірна платикуртичність, але γ₄ > −0.7)
- g₃_теор = 0.920 → теоретичне покращення 8.0%
- Bootstrap g₃_emp = 0.978 (PMM3 фактично = МНК, теорія не реалізується)
- LOO-MSE: PMM3/OLS = 0.999 (різниця < 0.1%)
- **Причина:** γ₄ = −0.431 не досягає емпіричного порогу γ₄ ≲ −0.7;
  оцінка kappa з залишків МНК додає шум, що нівелює теоретичний виграш.

Разом з іншими 4 наборами даних (γ₄ ∈ [−0.54, −0.12]) PMM3 не дає переваги над МНК: g₃_emp ∈ [0.97, 1.03]. Емпіричний поріг ефективності PMM3: **γ₄ ≲ −0.7**.

## 7. Висновки

1. **Теоретично**: PMM3 забезпечує від 7% до 95% зниження дисперсії порівняно з МНК для TN-похибок з λ ∈ {1, …, 5}. MLE-TN при правильній специфікації дає ще вищий виграш, але оцінює p+2 параметри.

2. **Монте-Карло**: MLE-TN значно ефективніший за PMM3 при λ ≥ 1.5 (якщо TN-специфікація правильна). Емпіричний g₃(PMM3) наближається до теоретичного при n ≥ 100–200.

3. **Робастність**: PMM3 завжди ARE ≈ 1 при misspecification; MLE-TN може бути гіршим за МНК. PMM3 є безпечним вибором при невизначеності щодо форми розподілу.

4. **Реальні дані**: Практична ефективність PMM3 підтверджена на iris versicolor (γ₄ = −0.966, n = 50): 43% зниження дисперсії, LOO-MSE краще на 2.3% vs МНК.

5. **Емпіричний поріг**: PMM3 > МНК лише при γ₄ ≲ −0.7. При −0.7 < γ₄ < 0 всі методи практично еквівалентні МНК.

6. **Практичні рекомендації**:
   - γ₄ ≲ −0.7 + впевненість у TN-формі + великий n → MLE-TN
   - γ₄ ≲ −0.7 + невизначеність щодо розподілу → PMM3
   - γ₄ > −0.7 → МНК (OLS)

## Анотація (English — для журналу)

We compare three estimators of linear regression parameters under symmetric
non-Gaussian errors following the Two-piece Normal (TN) distribution: Ordinary
Least Squares (OLS), Maximum Likelihood for TN (MLE-TN), and the third-degree
Polynomial Maximization Method (PMM3). MLE-TN is fully parametric and achieves
the Cramér–Rao bound under correct model specification, while PMM3 is
semi-parametric and relies only on sample moments of order two, four, and six,
without assuming a specific distributional form. We derive the theoretical
efficiency coefficient g₃ = Var(PMM3)/Var(OLS) = 1 − γ₄²/(6 + 9γ₄ + γ₆),
confirm asymptotic normality of PMM3, and verify its numerical stability via
Newton–Raphson convergence tests across unimodal and bimodal TN regimes.
A Monte Carlo study (M = 500, n ∈ {46, 100, 200, 500}, λ ∈ {0.5, …, 5})
shows that PMM3 achieves up to 95% variance reduction over OLS at λ = 5
(γ₄ = −1.85), and that ĝ₃ converges to the theoretical g₃ within ±10%
for n ≥ 100. Under model misspecification (uniform, triangular, logistic,
t₁₀ errors), PMM3 maintains ARE ≈ 1, while MLE-TN can lose efficiency.
Applied to Fisher's Iris versicolor dataset (n = 50, γ₄ = −0.966), PMM3
achieves 43% bootstrap variance reduction and a 2.3% LOO-CV improvement
over OLS. The empirical efficiency threshold is γ₄ ≲ −0.7. We recommend
PMM3 over MLE-TN when distributional form is uncertain, and over OLS
whenever γ₄ ≲ −0.7 is observed in regression residuals.

**Keywords:** polynomial maximization method; linear regression; two-piece
normal distribution; maximum likelihood; relative efficiency; robustness;
Monte Carlo simulation
